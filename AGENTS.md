# Agents

Korte instructie voor AI-agents (Claude Code, Codex, e.a.) die in deze repo werken.

Lees eerst:
- `START-HERE.md` — onboarding.
- `CLAUDE.md` — volledige context (positionering, schrijfstijl, techniek, valkuilen, status).
- `PRD.md` — takenlijst en aanbevolen volgorde.

Kort:
- Smart-Scale is een **AI-bedrijf** dat bedrijven helpt AI te implementeren. Websites zijn het
  hoofdaanbod. Noem het geen "AI-studio".
- Werk in `nieuwe-website/`. `oude-website/` is alleen referentie.
- Toon: kort, direct, menselijk, geen hype, **geen em-dashes**. Zeg gewoon wat je krijgt.
- **Alle asset-paden root-absoluut** (`/style.css`, `/images/...`), nooit relatief.
- Statische site, geen framework. Doel: Lighthouse 100/100/100/100.
- Lokaal draaien: `npm install && npm start` (of `python -m http.server` vanuit `nieuwe-website/`).
